## Regras de Negócio
1. Não é permitido cadastrar produtos com o nome vazio.
2. Não é permitido cadastrar produtos com o preço negativo.
3. O sistema deve calcular o valor total do pedido automaticamente à medida que os produtos são adicionados.
4. Um pedido só pode ser finalizado se tiver pelo menos um produto adicionado.
5. O sistema deve permitir consultar os pedidos de um determinado dia e calcular o faturamento total (soma de todos os pedidos) daquele dia.
